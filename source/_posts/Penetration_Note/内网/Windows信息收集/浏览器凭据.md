# HackBrowserData

[GitHub - moonD4rk/HackBrowserData: Extract and decrypt browser data, supporting multiple data types, runnable on various operating systems (macOS, Windows, Linux).](https://github.com/moonD4rk/HackBrowserData)

```shell
# 自动拖取全部
HackBrowserData.exe
#指定edge浏览器进行脱取其数据
HackBrowserData.exe -b edge -p 'C:\Users\User\AppData\Local\Microsoft\Edge\User Data\Default'
#指定edge的key浏览器进行脱取其数据
HackBrowserData.exe -b edge -p 'C:\Users\User\AppData\Local\Microsoft\Edge\User Data\Default' -k 'C:\Users\User\AppData\Local\Microsoft\Edge\User Data\Local State'
#指定chrome浏览器进行脱取其数据
HackBrowserData.exe -b chrome -p 'C:\Users\win10\AppData\Local\Google\Chrome\User Data\Default\Login Data'
#指定chrome的key浏览器进行脱取其数据
HackBrowserData.exe -b chrome -p 'C:\Users\User\AppData\Local\Microsoft\Edge\User Data\Default' -k 'C:\Users\win10\AppData\Local\Google\Chrome\User Data\Local State'
```

```shell
# 密钥地址
AppData\Roaming\Mozilla\Firefox\Profiles\随机字符.default-release\key4.db
```

不同版本的Firefox保存记录的文件名称不同

Version大于等于**32.0**，保存记录的文件为**logins.json  
**Version大于等于**3.5**，小于**32.0**，保存记录的文件为**signons.sqlite**

注意：Master Password 这个设置

# 360

[GitHub - hayasec/360SafeBrowsergetpass: 这是一个一键辅助抓取360安全浏览器密码的CobaltStrike脚本以及解密小工具，用于节省红队工作量，通过下载浏览器数据库、记录密钥来离线解密浏览器密码。](https://github.com/hayasec/360SafeBrowsergetpass)

注意：360浏览器是基于Chromium 86内核开发的。

```shell
# 360浏览器密码默认存储路径
## 360chrome 
C:\Users\win10\AppData\Local\360Chrome\Chrome\User Data\Default
## 360浏览器
C:\Users\win10\AppData\Roaming\360se6\User Data\Default
```

360安全浏览器离线解密:  
首先切到指定用户权限下，获取机器id，把assis2.db账密数据库文件拖回本地执行解密

```shell
reg query "HKLM\SOFTWARE\MICROSOFT\CRYPTOGRAPHY"/v"MachineGuid"
```
