# 域信息

```shell
#查询当前登录域及登录用户信息
net config workstation
```
```powershell
Import-Module .\PowerView.ps1;
Get-NetForest
```

# 查找域控

方法一：查找时间服务器

```shell
net time /domain
```

方法二：查看 domain controllers 组

```shell
net group "domain controllers" /domain
```

方法三：nslookup 查询

如果知道域名为 abc.com，则可以使用如下 nslookup 命令查询

```shell
nslookup -qt=ns abc.com
```

方法四：查询域控详细信息

```powershell
Import-Module .\PowerView.ps1;
Get-NetDomainController
```

# 查询域内用户

```shell
#查询域内用户
net group "domain users" /domain
net user /domain
#查询指定域用户信息
net user test /domain
#查询域用户详细信息
wmic useraccount get /all
```
```powershell
Import-Module .\PowerView.ps1;
#查询域内用户
Get-NetUser | select name
#统计域内用户数量
Get-NetUser | select name | measure | findstr Count
#查询域用户详细信息
Get-NetUser
#查询指定用户在哪些组中
Get-NetGroup -UserName test
```

# 查询域内主机

```shell
#这条命令只会查询 domain computers 组内的主机，并不包含域控等其他组内的主机
net group "domain computers" /domain
```
```powershell
Import-Module .\PowerView.ps1;
#查询域内主机，包括域控
Get-NetComputer
#统计域内主机数量
Get-NetComputer | measure | findstr Count
#查询带指定字符的主机
Get-NetComputer *AD*
```

导出域内主机详细信息(域控执行):

```shell
powershell "Get-ADComputer -Filter * | Select-Object Name,DistinguishedName,OperatingSystem,OperatingSystemVersion,ipv4Address|Export-CSV domain_computer.csv"
```

# 查询域内的组

```shell
#查询域内的全局组和通用组
net group /domain
#查找指定组含有哪些用户
net group "domain admins" /domain
```
```shell
Import-Module .\PowerView.ps1;
#查询域内的所有组
Get-NetGroup
#查询到指定字符的组
Get-NetGroup *admin*
#查询指定用户在哪些组中
Get-NetGroup -UserName test
```

# 查询域管理员

```shell
#查看域管理员,该组内的成员对域控拥有完全控制权
net group "domain admins" /domain
#查看企业管理组,该组内的成员对域控拥有完全控制权
net group "enterprise admins" /domain
```

## 查询用户 SID 和域 SID

```shell
#查看用户 SID 和域 SID，如用户的 SID 是：S-1-5-21-2189311154-2766837956-1982445477-520 则域 SID 则是去掉最后的 520
```
