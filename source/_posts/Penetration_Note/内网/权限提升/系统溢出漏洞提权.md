# CVE-2018-8120

<details open>
<summary>漏洞描述</summary>

Windows 操作系统 Win32k 的内核提权漏洞

部分版本 Windows 系统 win32k.sys 组件的 NtUserSetImeInfoEx()系统服务函数内部未验证内核对象中的空指针对象,普通应用程序可利用该空指针漏洞以内核权限执行任意代码
