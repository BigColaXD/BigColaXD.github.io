# crypto01

![image.png](https://cdn.nlark.com/yuque/0/2025/png/39170111/1758374687320-deacbb62-5bdd-497e-a92c-f826ad145164.png)

```plain
e = 23
n = 76198847820766755799575419083507924439717371587288270051268263988998008696707
c = 22544322401515359081098558097211421427974939372246104662667107456064368566176

p = 278397922222800446346367107995770769671
q = 273704800712503895893612643539504241317

# 在模p下求所有e次根
Rp = Zmod(p)(c).nth_root(e, all=True)
# 在模q下求所有e次根
Rq = Zmod(q)(c).nth_root(e, all=True)

# CRT
for rp in Rp:
    for rq in Rq:
        # 使用SageMath的CRT实现
        m = crt([Integer(rp), Integer(rq)], [p, q])
        try:
            # 将整数转换为字节
            flag = int(m).to_bytes((int(m).bit_length() + 7) // 8, byteorder='big')
            if flag.startswith(b'flag{'):
                print(flag.decode())
                break
        except:
            pass
    else:
        continue
    break
```
